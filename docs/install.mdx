#  Fitur Utama

- Autentikasi pengguna (login & register)
- Dashboard responsif
- Manajemen data real-time (Livewire)
- Komponen UI dengan Tailwind CSS
- Mode gelap otomatis

##  Tech Stack

- Laravel 10
- Livewire 3
- Tailwind CSS 3
- Alpine.js

##  Instalasi

```html
<input class="foo">
```


##   Konfigurasi .env

Pastikan untuk mengatur variabel berikut:

```dotenv
APP_NAME=MyApp
APP_URL=http://localhost:8000

DB_DATABASE=myapp
DB_USERNAME=root
DB_PASSWORD=
```

## Testing

```bash
php artisan test
```

### Role & Akses

| Role       | Fitur yang Bisa Diakses           |
|------------|-----------------------------------|
| Admin      | Semua fitur                       |
| User       | Dashboard, Profil, Transaksi      |
| Guest      | Halaman publik, Login & Register  |
